hexo官方文档

  1. hexo new post “blog title”
  2. hexo g //生成
  3. hexo s //本地服务器运行，通过localhost访问
  4. hexo d //部署

通过以下初始化归档和分类目录

  1. hexo new page “categories”
  2. hexo new page “tags”
  3. 打开 categories 文件夹下的 index.md ，在最下面一行加一行文字就行，注意中间有空格。type: categories;
  4. 打开 tags 文件夹下的 index.md ，在最下面一行加一行文字就行，注意中间有空格。type: tags;

通过配置_config.yml的excerpt设置文章的展开全文
